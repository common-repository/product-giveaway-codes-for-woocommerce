Batch generates unique coupon codes for product giveaways in WooCommerce.

== Description ==

The Product Giveaway Codes plugin facilitates single product giveaways in WooCommerce by batch generating unique giveaway codes that customers can enter at checkout to receive the product for free. It uses WooCommerce's built-in coupon functionality with product and usage limited 100% discounts to achieve this.

Features:

* Allow each code to be used one or more times.
* Set an expiry date for each batch of codes.
* Customize the giveaway code format.
* Batch delete giveaway codes when no longer needed.

== Installation ==

1. Click "Plugins" > "Add New" in the WordPress admin menu.
1. Search for "Product Giveaway Codes".
1. Click "Install Now".
1. Click "Activate Plugin".

Alternatively, you can manually upload the plugin to your wp-content/plugins directory.
